# 솔개톡 클라이언트
2019년 3학년 1학기 캡스톤디자인 솔개의선택

## PySide2(Qt) 설치 방법
- 명령 프롬프트에서 pip install PySide2 --user 입력하면 설치 완료

## PyMysql 설치 방법
- 명령 프롬프트에서 pip install PyMysql --user 입력하면 설치 완료
